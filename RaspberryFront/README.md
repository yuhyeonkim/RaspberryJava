RasberryFront is React + Semantic-UI implementation of Digital Signage for Hanium project. 


## Usage

Check client and server in 'examples' directory. 

To see it in action:

````bash
$ npm install
````

### Start server:
````bash
$ npm run-script example-server
````

### Start client:
````bash
$ npm run-script example-client
````
